
  # Variant Setup (Copy)

  This is a code bundle for Variant Setup (Copy). The original project is available at https://www.figma.com/design/GRfnD5GZXufUDMIAMnmWiu/Variant-Setup--Copy-.

  ## Stack

  This project now targets a modern Vite frontend setup with React 19 and TypeScript project references while preserving the existing Figma-exported structure and custom asset resolver.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  Run `npm run build` to create the production build in `build/`.

  Run `npm run preview` to preview the production build locally.
